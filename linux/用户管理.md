## 用户管理

# 用户管理的重要配置文件

| 配置文件     | 内容                                               |
| ------------ | -------------------------------------------------- |
| /etc/passwd  | 用户名 密码位 用户编号 归属组编号 姓名 $HOME Shell |
| /etc/shadow  | 用户名 已加密密码 密码改动信息 密码策略            |
| /etc/group   | 群组名 密码位 群组编号 组内用户                    |
| /etc gshadow | 群组密码相关文件                                   |
| /etc/sudoers | 用户名 权限定义 权限                               |
